## Features
* Player/Scav ESP
* Items
* Grenades
* Extracts
* Aimbot
* Flyhack
* No Recoil/Spread/Sway
* Unlimited Stamina
* Thermal vision toggle
* Night vision toggle
* No visor
* Loot through walls
* Moon jump (jump like you are on the moon)
* FlyHack
* Fast Load Mags
* Fast Un-Load Mag
